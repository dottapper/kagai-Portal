## 🌟 主な機能

### 📰 News & イベントページ
- **インタラクティブカレンダー** - 月間ナビゲーションとイベント表示
- **イベント詳細表示** - クリックでポップアップ表示
- **ページネーション** - ニュース記事の効率的な閲覧

### 📢 プレスリリースページ
- **カテゴリフィルター** - お知らせ、イベント発表、メディア情報、受賞・表彰
- **詳細モーダル** - プレスリリースの詳細表示
- **報道関係者向けコンタクト** - 取材申し込みフォーム

### 📖 花街・芸者ガイドページ
- **ステップバイステップガイド** - 段階的な学習システム
- **プログレス表示** - 学習の進捗確認
- **FAQ セクション** - よくある質問と回答

### 🎭 花街・芸者文化紹介ページ
- **タブナビゲーション** - 日本舞踊、音楽、茶道、華道、着物・装身
- **詳細な文化解説** - 各分野の歴史と技法
- **季節の特色** - 四季折々の文化的特徴

### 🌍 海外の方向けページ
- **完全バイリンガル対応** - 日本語・英語の切り替え機能
- **文化的配慮** - 外国人向けの詳細な説明
- **体験プラン** - 価格帯別の体験オプション

### 📚 花街・芸者用語集ページ
- **リアルタイム検索** - 即座に用語を検索
- **50以上の専門用語** - 包括的な用語データベース
- **詳細説明モーダル** - 各用語の詳しい解説

## 🎨 デザイン特徴

- **明朝体フォント** - Noto Serif JPによる上品な表示
- **完全レスポンシブ** - モバイル、タブレット、デスクトップ対応
- **モダンデザイン** - 洗練されたUI/UX
- **アクセシビリティ** - ARIA属性、キーボードナビゲーション対応
- **パフォーマンス最適化** - 高速読み込みと滑らかなアニメーション

## 🔧 技術仕様

- **HTML5** - セマンティックマークアップ
- **CSS3** - Grid Layout、Flexbox、CSS Variables
- **Vanilla JavaScript** - フレームワークなしの高速実装
- **レスポンシブデザイン** - Mobile-First アプローチ
- **Web Standards** - W3C準拠

## 🚀 使用方法

1. `index.html` をブラウザで開く
2. メニューから各ページにナビゲート
3. インタラクティブな機能を体験
